# M10.7 Session Report and Separation Scoring

Solo training now has a live Session Report that replaces the old approach-only report. The report is available from **Scenario > Session Report** and refreshes while the scenario is running.

## Implemented Slice

- Live student-facing scorecard with score, grade, coaching notes, active issues, event timeline, approach scores, and runway statistics.
- FAA-grounded airborne separation scoring for:
  - IFR radar separation using 7110.65 §5-5-4 / §5-5-5 and vertical minima from §4-5-1.
  - Class B VFR separation using 7110.65 §7-9-4.
  - Visual-follow suppression when traffic has been reported in sight and a follow instruction is active.
- FAA-grounded same-runway separation scoring for actual runway-operation trigger points:
  - Departures starting takeoff roll behind departures or landings using 7110.65 §3-9-6(a)-(b).
  - Arrivals crossing the landing threshold behind departures or landings using 7110.65 §3-10-3(a).
  - SRS Category I/II/III distances from FAA ACD `Srs`, with fallback to the §3-9-6 note definitions when the database has no category.
  - Live Session Report rows show plain-language required and actual runway-separation evidence for these events.
- FAA-grounded wake scoring for actual runway/approach trigger points:
  - Departure wake intervals from 7110.65 §3-9-6(f)-(g) and intersection-departure intervals from §3-9-7(a).
  - CWT mile-based separation from 7110.65 §5-5-4(g) can satisfy the departure wake requirement when it is already tighter than the time interval.
  - Terminal same-runway and close-parallel approach spacing uses 7110.65 §5-5-4(h) Table 5-5-2.
- Training-only terminal warnings labeled separately from controller-authentic CRC conflict alerts.

## Follow-Up Scope

- Traffic-advisory scoring from 7110.65 §2-1-21 and §7-9-5 once command history can reliably prove advisory issuance.
- Reciprocal/opposite-direction operations and intersecting-runway scoring.
- Projected-flight-path crossing wake intervals, facility wake waivers, and wake-advisory phraseology proof.
