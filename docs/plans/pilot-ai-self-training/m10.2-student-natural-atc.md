# M10.2 — Student speaks/types real ATC

**Status:** implemented
**Size:** Small (~3–5 days)
**Maturity:** shipped in code — revalidated before implementation on 2026-05-07
**Commit:** `85196100 feat: solo training mode natural ATC input`

## Context

After M10.1 (and now M10.1.1+), the simulator emits realistic readbacks but the student still has to type canonical RPO commands (`N123AB DM 5000`). M10.2 lets the student speak (or type) natural-language ATC English, parsed back into canonical via the existing `PhraseologyMapper` + `LocalLlmCommandMapper` fallback. This is the smaller of the two "feels alive" pieces; M10.3 adds audible voice on top.

## What ships

The student-controller's input box accepts natural-language ATC, not just canonical. The typed path first tries the normal command parser. If parsing fails and session `SoloTrainingMode` is enabled, it runs the same transcript mapping stage used by PTT (`PhraseologyCommandMapper`, with LLM fallback only when speech/LLM is enabled and configured), validates the mapped canonical command, and dispatches it once.

`SoloTrainingMode` is now a persisted client default and a synchronized session setting. The loading RPO's preference is pushed after scenario load; any RPO can change the active session value from the command-bar session flyout.

## Open question

Should `SoloTrainingMode` also turn on M10.1's auto-readback? Answer for this implementation: **no implicit coupling** beyond the sim behavior already gated by `SimScenarioState.SoloTrainingMode`. The user-facing setting is still off by default and session-scoped once a room is active.

## Riskiest part

None of `PhraseologyMapper`'s rules are scoped to "controller-side" vs "pilot-side." For v1 this intentionally stays lenient: the ground rules already accept instructor/RPO/pilot colloquialisms such as natural taxi phrasing and optional runway words. The fallback validates the mapped command through the canonical parser before dispatch, so unmapped partial transcripts are surfaced as parse errors instead of being sent.
