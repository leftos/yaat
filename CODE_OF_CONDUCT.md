# Code of Conduct

YAAT adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is canonical at that link; please read it.

In short: be respectful, assume good faith, focus on the work. This applies everywhere project participation happens — GitHub issues and pull requests, Discussions, the Discord, and any other YAAT channel.

## Reporting

If you experience or witness behavior that violates the Code of Conduct, email **leftos@gmail.com**. Reports are handled privately by the maintainers listed in [MAINTAINERS.md](MAINTAINERS.md).

When reporting, include what happened, where, when, and who was involved if you're comfortable sharing. Anonymous reports are welcome but harder to act on.

## Enforcement

Maintainers are responsible for clarifying and enforcing these standards. Possible responses range from a private warning to a temporary or permanent ban from project spaces, depending on the situation.

Maintainers who do not follow or enforce the Code of Conduct in good faith may face the same consequences as anyone else, decided by the remaining maintainers.

## Scope

The Code of Conduct applies to all project spaces and also when someone is representing the project in public — for instance, on social media, at events, or in other VATSIM venues while identifiably wearing the YAAT hat.
